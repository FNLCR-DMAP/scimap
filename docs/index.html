Welcome to the scimap-dmap Conda Channel

This page hosts Conda packages for the scimap conda creation project under the DMAP of NCI.

The package dependencies are modified to fit the need of SPAC package and usage on NIDAP

Use this channel with the following command:

conda config --add channels https://fnlcr-dmap.github.io/scimap/

Available Platforms

  • Linux 64-bit
  • Noarch (Not Available)
  • OSX 64-bit (Not Available)
  • Windows 64-bit (Not Available)
